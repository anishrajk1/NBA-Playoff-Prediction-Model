# NBA Playoff Prediction Model

Machine learning project that predicts NBA playoff game outcomes using historical playoff and team performance data from the NBA API.

## Features
- Automated NBA data collection using nba_api
- Team strength dataset generation
- Logistic regression prediction model
- Team-vs-team playoff outcome prediction
- Feature engineering using season statistics

## Technologies
- Python
- Pandas
- Scikit-Learn
- NBA API
- Matplotlib
- Seaborn

## Model Performance
- Achieved ~58% prediction accuracy on unseen playoff matchup data
